# auto_ss

## 概要

設定した撮影間隔で自動的にスクリーンショットを実行するツール。

## 開発

|名称|バージョン|
|---|---|
|Python|3.10.8|
|PySide6|6.4.0|
|pyinstaller|5.6.2|

## ビルドコマンド

auto_ssディレクトリ内に移動し、以下のコマンドを実行。

### ・Windows

```shell
pyinstaller autoSS.spec
```

## ソース実行コマンド

auto_ssディレクトリ内に移動し、以下のコマンドを実行。

```shell
python auto_ss.py
```
